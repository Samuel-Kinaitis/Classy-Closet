Purpose

Please note that this website is a mock-up eCommerce platform created for demonstration purposes only.The goal of this website is to showcase the design and functionality of an eCommerce store.

Navigation and Menus

Use the header navigation menu above and footer menu below to navigate the website

Product Overview

Click on the desired product to view information about the product

Cart Overview

Click on the "X" button to remove products from the cart

Employee Overview

The Employee page allows for products to be modified or added to the website. Items can be removed, added, and promoted in either the featured and trending section.

How to Reset Website

Clearing cookies and site data via the the site information button in the search bar will reset the site to it's orginal state
